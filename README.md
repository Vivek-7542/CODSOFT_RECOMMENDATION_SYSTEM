# 🎬 Movie Recommendation System

A Python-based Movie Recommendation System that suggests movies to users based on their preferences using Machine Learning techniques. This project utilizes collaborative filtering with the K-Nearest Neighbors (KNN) algorithm to recommend similar movies.

---

## 📌 Features

* Recommend movies based on user preferences
* Uses Collaborative Filtering
* K-Nearest Neighbors (KNN) algorithm
* Data preprocessing with Pandas and NumPy
* Easy-to-understand code structure
* Suitable for beginners learning Recommendation Systems

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

---

## 📂 Dataset

This project uses the MovieLens dataset containing:

* Movies information
* User ratings
* Movie IDs and titles

Dataset Source: [MovieLens Dataset](https://grouplens.org/datasets/movielens/?utm_source=chatgpt.com)

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/movie-recommendation-system.git
```

2. Navigate to the project directory:

```bash
cd movie-recommendation-system
```

3. Install required libraries:

```bash
pip install pandas numpy scikit-learn matplotlib seaborn
```

---

## ▶️ Usage

Run the Python file:

```bash
python recommendation_system.py
```

Enter a movie name and the system will recommend similar movies based on user ratings.

---

## 📊 How It Works

1. Load movie and rating datasets.
2. Clean and preprocess data.
3. Create a user-movie matrix.
4. Apply K-Nearest Neighbors (KNN).
5. Find movies with similar rating patterns.
6. Display recommended movies.
